Comment installer EraDUCKation?

1. Il faut avoir la SDL d’installée : SDL2_ttf-2.0.4.14 ,SDL2_image-2.0.3,SDL2-2.0.8. 
2. Etre sous Linux
3. Exécuter le makefile dans un terminal
4. Lancer l’executable générer (eraduckation)
5. Jouer


Comment utiliser EraDUCKation?

Après le lancement du programme, une page SDL apparait.
Pour lancer le jeu, cliquez sur le bouton play, le jeu se lance sur votre terminal.
Une fois sur terminal, la saisie se fait au clavier.
Il faut :
	- choisir de jouer en local ou réseau
		-> en local : 
			*choisir solo ou multi
			*saisir le pseudo
			*choisir la difficulté
		-> en réseau:
			*choisir si on veut être client ou serveur
			*Si on est serveur:
				- choisir le numéro de port
			*Si on est client:
				- saisir le numéro de port
				- saisir l’IP du serveur
			*Choisir si on veut être méchant ou gentil

	- choisir de jouer une partie chargée
	- choisir de quitter le jeu
